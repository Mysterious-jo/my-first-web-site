CSS

Cascading style sheets are style sheet language or style language that describes how the markup language is actually displayed,[1] and is commonly used in HTML and XHTML, and can also be used in XML. It is a W3C standard, and it has a high degree of freedom in defining layouts and styles. The default filename[2] is style.css. If the markup language (ex: HTML) is responsible for the body of the website, CSS is responsible for decorating it like clothes and accessories. In other words, you can leave the HTML structure intact and just change the CSS file to make it look like a completely different website. In the case of CSS3, which is currently under development, functions such as shadow effects, gradients, and transformations have been added to replace images created with graphics editing programs. In addition, various animation features have been added, replacing Adobe Flash to some extent.
